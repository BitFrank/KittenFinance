FAQ:

1. Note KF is not a yield farming project. The pools are for distributing the governance token KIF. YFI only had 1 week of farming.

2. You can download MetaMask app to use KittenFinance on mobile devices. The url is www.kitten.finance (remember the www).

3. The first deposit to a pool needs an "approval" tx. Wait for it to confirm, then refresh the page to stake to the pool. 

4. You can directly trade KIF/WETH or provide KIF+WETH liquidity on UniSwap. This actually saves gas, so there's no need to unwrap WETH. To manually swap between WETH and ETH, click here. The rate is always 1 ETH = 1 WETH. Remember to leave some ETH for paying gas.

5. NEVER NEVER set a high "slippage tolerance %" when trading on Uniswap, or bots will frontrun you. Better use 0.1% or 0.5% slippage and high gas. Click the top-right button in Uniswap to set it. That 0.5% is the extra slippage that can occur if someone frontruns you. So if you set the slippage tolerance too high, bots will be happy to do that.

6. About the pools. Pool 1 has zero risk and lowest return. If KIF remains stable or grows a bit / drops a bit, then pool 3 is better than pool 2. If KIF rises a lot then pool 2 is better. Moreover pool 3 can be used to accumulate KIF when it drops. If you want more KIF, go to pool 3 when you think KIF is overvalued, and go to pool 2 when you think it's undervalued. So, the selection of pools depends on your view of future KIF prices. It might be a good idea to diversify.
